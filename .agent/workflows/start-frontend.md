---
description: How to start the Frontend Application for Mens clothing store.
---

To get the login and signup features working, you must have the Python backend running:

1. Open a terminal.

2. Navigate to the frontend directory:
 bash
   cd frontend
   
3. Start the server:
 bash
   flutter run -d chrome

|||Recommended|||
use this command if the backend is not running
flutter run -d chrome --web-browser-flag "--disable-web-security"
